# Job Action Accessibility Design

## Goal

親子ジョブに同じページ番号が存在しても再試行対象を識別でき、主要な再開・再試行操作を44px以上のヒット領域で利用できるようにする。

## Design

- 子ページの再試行ボタンは、親ジョブの日本語種別、親ジョブID、ページ種別と番号をARIAラベルに含める。
- 親ジョブ再開、子ページ再試行、文書詳細のページ再試行は、既存button自体へ `min-h-11 min-w-11` を付ける。
- 文字、アイコン、既存の高さ指定は維持し、最小寸法だけを加えてコンパクトな見た目を保つ。
- RTLでARIAラベルと3種類のボタンの最小寸法クラスを検証する。
